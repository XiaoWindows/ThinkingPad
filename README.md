# ThinkingPad
这是用java编写的随机问题程序，你可以利用它实现自我提升或者培养批判性思维。
运行环境：Java23
程序功能：
1.导入问题 你可以导入任何你想的问题，这样就可以实现高度自定义。
2.随机问题（核心） 点击程序里的刷新按钮或者问题框，就可以刷新问题，问题是随机的，这样有利于培养批判性思维，确保思维不会跟着你的大脑走。
3.保存历史 你可以保存你的思维记录，这样在查看历史记录中就可以随时查看你思考过的痕迹。
4.查看导入问题 想查看导入了什么问题吗？点击左上角的选项卡就可以查看导入了什么问题。
5.全屏 想让思维更沉静？试试打开全屏吧。
6.背景图片支持及切换背景图片 这个程序的图片是保存在文件目录下的  image文件夹  中，如果没有图片，则会使用默认背景，如果有两张及以上图片，试试点击刷新，它会自动切换的。

ThinkingPad，不一样的思维空间。
